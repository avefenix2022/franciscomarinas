Navegadores y Editores

Navegadores HTML

Los navegadores web son aplicaciones que interpretan código HTML, CSS y JavaScript y lo renderizan en la ventana de la aplicación.

Editores HTML

Los editores HTML son editores de texto con los que escribimos codigo HTMl, CSS y Javacript y me permite guardar dichos archivos con extensiones .html, .css y .js. Basta con un Block de notas para escribir nuestros archivos HTMl, CSS y Javacript.

Márgenes superior e izquierdo de los navegadores

Los navegadores por defecto presentan unos márgenes superior e izquierdo entre el contenido de la página web y su propia ventana.

El consorcio W3C publicó en su recomendación CSS 2.11 del 23 de abril de 2009, un valor por defecto para cada propiedad de CSS que deberían adoptar los navegadores. Algunos navegadores hicieron caso omiso a sus indicaciones por lo que el valor de estos margenes varía de un navegador a otro.

Espacios

El navegador solo mostrará un espacio aunque en el editor HTML se escriban más de uno.

Retornos de carro

El navegador no mostrará los retornos de carro que se escriben en el editor HTML.

Caso curioso: Retorno de carro en los editores Web para elementos en línea.

Un detalle curioso es el resultado de la utilización de retornos de carro en el editor web para organizar el código, que con elementos en línea el resultado del renderizado en el navegador web introduce un espacio entre estos elementos en línea.

See the Pen Retorno de carro en los editores Web par elementos en línea by Francisco (@franciscomarinas) on CodePen.
