CSS Grid Layout

CSS4 como tal no existe. A partir de CSS3 el estándar para hojas de estilo en cascada se divide en módulos, cada uno con su número de versión. Dentro de estos nuevos módulos CSS4 nos encontramos con Grid Layout. No sustituye a Flexbox, si no que lo complementa y es una alternativa para el mismo problema.

  1. CSS Grid Layout

Usar CSS Grid Layout

Definiendo un grid

Para definir una cuadrícula se utiliza display: grid o display: inline-grid en el elemento padre. A continuación, puede crear una cuadrícula utilizando las propiedades grid-template-columns y grid-template-rows.

La propiedad grid-gap para crear una separación entre columnas y filas de 10px. Esta propiedad es una abreviatura para grid-column-gap y grid-row-gap para que pueda establecer estos valores individualmente.

Todos los hijos directos del padre ahora se convierten en elementos de la cuadrícula y el algoritmo de colocación automática los establece uno en cada celda de la cuadrícula. Se crearán filas adicionales según sea necesario.

copy 

<div class="wrapper">
  <div class="box a">A</div>
  <div class="box b">B</div>
  <div class="box c">C</div>
  <div class="box d">D</div>
  <div class="box e">E</div>
  <div class="box f">F</div>
</div>
copy 

body {
    margin:                40px;
}

.wrapper {
    display:               grid;
    grid-template-columns: 100px 100px 100px;
    grid-gap:              10px;
    background-color:      #fff;
    color:                 #444;
}

.box {
    background-color:      #444;
    color:                 #fff;
    border-radius:         5px;
    padding:               20px;
    font-size:             150%;
}

La unidad fr representa una fracción del espacio libre disponible.

La unidad fr (o <flex>) se define como una fracción del espacio disponible. La cuota de cada columna o fila del espacio libre se puede calcular como:
<flex> * <espacio libre> / <suma de todos los factores flex> de la columna o de la fila
Eso es de acuerdo con el W3C.

Es más fácil pensar en términos de agregar todas las unidades fr que utiliza a lo largo de un eje y luego dividir la cantidad de espacio libre que se asignará entre los elementos que utilizan estas unidades. Se podría tener un elemento con 3fr ocupando tres veces la cantidad de espacio que un hermano con 1fr asignado. Si sólo se definieran estos dos elementos con la unidad fr en ese eje, el espacio disponible total se calcularía dividiéndolo en cuatro, con tres partes iguales asignadas al primer elemento y la parte igual restante asignada al segundo.
