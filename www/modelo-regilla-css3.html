CSS Grid Layout

CSS4 como tal no existe. A partir de CSS3 el estándar para hojas de estilo en cascada se divide en módulos, cada uno con su número de versión. Dentro de estos nuevos módulos CSS nos encontramos con Grid Layout. No sustituye a Flexbox, si no que lo complementa y es una alternativa para el mismo problema.

  1. CSS Grid Layout

Conceptos básicos del Módulo de Cuadrícula

Definiendo una cuadrícula

Introduce un sistema de cuadrícula bidimensional y define la relación en términos de:.

  • tamaño
  • posición y
  • capa

¿Qué es una cuadrícula?

Conjunto de intersección de líneas horizontales y verticales que definen filas y columnas (pistas). Los elementos se pueden colocar en la cuadrícula, respetando esas columnas y filas. El diseño de la cuadrícula CSS tiene las siguientes características:

Tamaño de pista fijo y flexible

El tamaño de pista se puede establecer en tamaño fijo con px o tamaños flexibles utilizando % o la unidad fr (flex).

Colocación del elemento

Puede colocar elementos en una ubicación precisa en la cuadrícula usando números de línea, nombres o apuntando a un área de la cuadrícula. Hay un algoritmo para controlar la ubicación de los elementos que no tienen una posición explícita en la cuadrícula.

Creación de pistas adicionales para el contenido

La especificación también se ocupa del contenido agregado fuera de una cuadrícula declarada, que agrega filas y columnas adicionales cuando sea necesario. Se incluyen características para agregar "tantas columnas como las que caben en un contenedor".

Control de alineación

La cuadrícula contiene características de alineación para controlar cómo se alinean los elementos una vez ubicados en un área de la cuadrícula, y cómo se alinea toda la cuadrícula.

Control de contenido superpuesto

Se puede colocar más de un elemento en una celda o área de la cuadrícula, pueden superponerse parcialmente entre sí. Esta estratificación se puede controlar con la propiedad z-index.

Grid es una poderosa especificación y cuando se combina con otras partes de CSS como flexbox, puede ayudarte a crear diseños que antes eran imposibles de construir en CSS.

Todo comienza creando una cuadrícula en su contenedor.

Contenedor de cuadrícula

Al crear un contenedor de cuadrícula declarando display: grid o display: inline-grid todos los hijos directos de ese elemento se convertirán en elementos de la cuadrícula.

En el siguiente ejemplo, hay un div que contiene un contenedor y dentro cinco elementos secundarios.

copy 

<div class="contenedor">
  <div class="celda">1</div>
  <div class="celda">2</div>
  <div class="celda">3</div>
  <div class="celda">4</div>
  <div class="celda">5</div>
</div>
copy 

/* Reset.css incluido */

.contenedor {
  display:    grid;
}

.celda {
  background: #abcdef;
  height:     2em;
  margin:     .2em;
  color:      #fff;;
}

See the Pen Contenedor cuadrícula by Francisco (@franciscomarinas) on CodePen.

Todos los hijos directos ahora son elementos de la cuadrícula. En un navegador web, no verá ninguna diferencia en cómo se muestran estos elementos antes de convertirlos en una cuadrícula, ya que la cuadrícula ha creado una regilla de columna única para los elementos.

Pistas de cuadrícula

Definimos filas y columnas en nuestra cuadrícula con las propiedades grid-template-columns y grid-template-rows. Estos definen pistas de cuadrícula. Una pista de cuadrícula es el espacio entre dos líneas de la cuadrícula.

Al ejemplo anterior agregar la propiedad grid-template-columns.

Se he creado una cuadrícula con tres pistas de columna de 200 píxeles de ancho. Los elementos secundarios se distribuirán en esta cuadrícula uno en cada celda de la cuadrícula.

copy 

<div class="contenedor">
  <div class="celda">1</div>
  <div class="celda">2</div>
  <div class="celda">3</div>
  <div class="celda">4</div>
  <div class="celda">5</div>
</div>
copy 

/* Reset.css incluido */

.contenedor {
  display:               grid;
  grid-template-columns: 200px 200px 200px;
}

.celda {
  background:            #abcdef;
  height:                2em;
  margin:                .2em;
  color:                 #fff;;
}

See the Pen Pistas de cuadrícula by Francisco (@franciscomarinas) on CodePen.

Unidad fr

La unidad fr representa una fracción del espacio libre disponible.

La unidad fr (o <flex>) se define como una fracción del espacio disponible. La cuota de cada columna o fila del espacio libre se puede calcular como:
<flex> * <espacio libre> / <suma de todos los factores flex> de la columna o de la fila.

Se podría tener un elemento con 3fr ocupando tres veces la cantidad de espacio que un hermano con 1fr asignado.

El siguiente ejemplo crea tres columnas iguales ocupando todo el espacio disponible:

copy 

<div class="contenedor">
  <div class="celda">1</div>
  <div class="celda">2</div>
  <div class="celda">3</div>
  <div class="celda">4</div>
  <div class="celda">5</div>
</div>
copy 

/* Reset.css incluido */

.contenedor {
  display:               grid;
  grid-template-columns: 1fr 1fr 1fr;;
}

.celda {
  background:            #abcdef;
  height:                2em;
  margin:                .2em;
  color:                 #fff;;
}

repeat()

Las cuadrículas grandes con muchas pistas pueden usar la notación repeat () para repetir una o todas las secciones de la lista de pistas. Por ejemplo, la definición de la grilla:

grid-template-columns: 100px 100px 100px;

@media(max-width:350px){ grid-template-columns: 1fr;}

copy 

.contenedor {
  display:               grid;
  grid-template-columns: 1fr 1fr 1fr;
}

Puede ser escrito como:

copy 

.contenedor {
  display:               grid;
  grid-template-columns: repeat(3, 1fr);
}

En el siguiente ejemplo, es una cuadrícula con una pista inicial de 20 píxeles, luego una sección repetitiva de 6 pistas de 1fr y luego una pista final de 20 píxeles.

copy 

.contenedor {
  display: grid;
  grid-template-columns: 20px repeat(6, 1fr) 20px;
}

En este sigiente ejemplo, la regilla constará de 10 pistas. Este patrón se repetirá cinco veces: una pista de 1 fr, y luego una pista de 2 fr.

copy 

.contenedor {
  display: grid;
  grid-template-columns: repeat(5, 1fr 2fr);
}
